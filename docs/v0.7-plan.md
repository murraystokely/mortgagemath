# v0.7.0 - International & Flexible Repayment

Status: **planning.**

## Scope

v0.7.0 adds support for common international and commercial repayment
structures that fall outside the standard monthly amortizing annuity
model, but remain within the library's cent-accurate mandate.

## Features

### 1. Zero-Interest Support

Support for loans with `annual_rate=0`.

- **Implementation:** Update `periodic_payment` and `amortization_schedule`
  to handle the case where the periodic rate is zero. The payment becomes
  a simple division: `principal / total_payments`.
- **Validation:** Use a promotional 0% financing example or a synthetic
  fixture.

### 2. Swedish Serial Loans (*Rak amortering*)

Support for constant-principal (serial) amortization where the total
payment decreases over time.

- **Implementation:**
  - Add `AmortizationType` enum: `ANNUITY` (default) and `SERIAL`.
  - Add `amortization_type` to `LoanParams`.
  - Update `amortization_schedule` to implement the serial loop:
    `principal_per_period = principal / total_payments` (rounded),
    `interest = balance * rate`, `payment = principal + interest`.
- **Validation:** Swedish *rak amortering* example (e.g., 1.2M SEK at 4%
  over 20 years).

### 3. Interest-Only (IO) Periods

Support for a period of interest-only payments before amortization
begins.

- **Implementation:**
  - Add `interest_only_months` (int) to `LoanParams`.
  - Update `amortization_schedule` to produce `interest_only_months / (12/ppy)`
    interest-only rows before starting the amortizing section.
  - Recast the amortizing section over the remaining term.
- **Validation:** CFPB sample disclosure for an Interest-Only loan
  ($211,000 / 4% / 5-yr IO period).

### 4. Canadian Accelerated Bi-Weekly

Convenience constructor for the common Canadian "accelerated" cadence.

- **Implementation:**
  - Add `canada_accelerated_biweekly` to `_constructors.py`.
  - Logic: compute the standard monthly payment, divide by 2, and set as
    a bi-weekly `payment_override`.
- **Validation:** RBC worked example ($350,000 at 5% over 25 years).

## Technical Changes

### `mortgagemath/_types.py`

- Add `AmortizationType(Enum)` with `ANNUITY` and `SERIAL`.
- Add `amortization_type: AmortizationType = AmortizationType.ANNUITY` to `LoanParams`.
- Add `interest_only_months: int = 0` to `LoanParams`.

### `mortgagemath/_payment.py`

- Update `periodic_payment` to handle `r=0`.
- If `r=0`, return `(loan.principal / loan._amort_payments).quantize(_PENNY, rounding=rounding)`.

### `mortgagemath/_schedule.py`

- Handle `interest_only_months` in the schedule loop.
- Implement the `SERIAL` amortization path.

### `mortgagemath/_constructors.py`

- Add `canada_accelerated_biweekly`.

## Out of Scope for v0.7.0

- Offset mortgages (requires external balance schedule).
- Negative interest rates (economic anomaly, but mathematically possible).
- Multi-tranche loans.
